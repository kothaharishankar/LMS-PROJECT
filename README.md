# 📚 Learning Management System (LMS)

A web-based Learning Management System (LMS) designed to simplify and enhance the online learning experience. The platform provides separate interfaces for administrators, teachers, and students, enabling efficient management of courses, attendance, assignments, and academic activities.

## ✨ Features

- 👨‍🎓 Student Login and Dashboard
- 👨‍🏫 Teacher Login and Dashboard
- 📖 Course Management
- 📝 Assignment Management
- 📊 Attendance Tracking
- 📈 Student Performance Monitoring
- 🔐 Secure User Authentication
- 📱 Responsive User Interface
- 🎯 Easy Navigation and User-Friendly Design
